
# 🏏 Ball Classification using Decision Tree

This project demonstrates a simple machine learning case study using a Decision Tree Classifier from **scikit-learn**. The goal is to classify balls as either a **tennis ball** or a **cricket ball** based on basic features.

## 📊 Dataset

* Total samples: **15**
* Features:

  * Weight of the ball
  * Surface type (Rough = 1, Smooth = 0)
* Labels:

  * Tennis Ball = 1
  * Cricket Ball = 2

## ⚙️ Approach

* Data is manually encoded and split into training and testing sets.
* A **Decision Tree Classifier** is trained on the dataset.
* The model predicts the type of ball based on input features.

## ▶️ Example

Input: `[35, 1]`
Output: `Tennis Ball`

## 📌 Conclusion
This case study shows how basic feature-based classification can be implemented using Decision Trees, making it a good starting point for beginners in machine learning.
